## ✨ Destaques

- 🎧 **Design imersivo** focado na experiência do produto
- 🎬 **Animações CSS** fluídas e modernas
- 🛒 **Vitrine de produtos** com visual premium
- 📱 **100% responsivo** — mobile-first
- ⚡ **Zero dependências** — HTML, CSS e JS puros
- 🎨 **Dark theme** com paleta sofisticada

## 💡 Por que este projeto?

Desenvolvido para demonstrar que é possível criar uma experiência de e-commerce visualmente poderosa sem frameworks — apenas com HTML semântico, CSS avançado e JavaScript vanilla. Ideal para entender fundamentos antes de migrar para React ou Next.js.

## 🛠️ Tecnologias

- **HTML5** — Estrutura semântica
- **CSS3** — Animações, Grid, Flexbox, variáveis CSS
- **JavaScript** — Interatividade e DOM manipulation

## 🚀 Como rodar

```bash
# Clone o repositório
git clone https://github.com/MysterySalsicha/drop-headphones-loja-fones-premium.git

# Abra o projeto
cd drop-headphones-loja-fones-premium

# Abra o index.html no navegador
# (ou use Live Server no VS Code)
```

Não precisa de `npm install` — é puro HTML/CSS/JS! ✅
